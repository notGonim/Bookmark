### Installation

1. **_Clone the repository_**

```sh
$ git clone https://github.com/notGonim/Bookmark.git
```
2. **_Navigate to repository directory_**
```sh
$ cd Bookmark
```

3. **_Install dependencies_**

```sh
$ npm install
```


### Running

```sh
$ npm start
```
